The Chemistry Tools

This is a tool that has been created using Python as a programming language. This is a small way to help the scientific community. Through this one can enter the exciting world of chemistry through the base of chemistry using periodic table and also calculate several concentration terms and most importantly moles. This is easy to use software developed especially so that even a beginner can use it smoothly. This would be a great help to various chemistry labs in schools. This software has been designed to cut short the hassles of memorizing periodic table and save you from several calculations. This has been created in accordance to the latest IUPAC suggestions.
