## Features

- Easy to use
- Hovering effect
- Fullscreen mode
- Cross platform

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/aloner-pro/Chemistry-tools.git
```

Go to the project directory

```bash
  cd Chemistry-tools
```
Good to go just you need to have python.
